# Day 8: Enable Stop Protection for EC2 Instance

As part of the migration, there were some components added to the AWS account. Team created one of the EC2 instances where they need to make some changes now.

There is an EC2 instance named **`datacenter-ec2`** under **`us-east-1`** region, enable the **`stop protection`** for this instance.

Step 1: Go to EC2 Dashboard and Click on `Instances running` in resource section.

<img width="719" height="239" alt="image" src="https://github.com/user-attachments/assets/e40a9874-9629-4dd7-a864-e8f6c4cd3d73" />

Step 2: Select instance and Click on `Action` and from  `instance Settings`, click on `Change Stop Protection`.

<img width="1365" height="615" alt="image" src="https://github.com/user-attachments/assets/da747b15-192e-4642-a823-72a23b5d5ca6" />

Step 3: Then click on `Enable` and `Save`
